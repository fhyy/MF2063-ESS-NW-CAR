# Ultrasonic sensors
## features
Link to the sensor's [datasheet](https://www.mouser.com/ds/2/813/HCSR04-1022824.pdf).
* 5V power supply
* max range: 4m
* min range: 2cm

## Pin mappings
Left sensor: (D for digital pins)
Arduino|Sensor
-------|-------
D4|Trig
D5|Echo

Middle sensor:
Arduino|Sensor
-------|-------
D6|Trig
D7|Echo

Right sensor: 
Arduino|Sensor
-------|-------
D8|Trig
D9|Echo
